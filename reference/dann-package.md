# dann: Discriminant Adaptive Nearest Neighbor Classification

Implements Discriminant Adaptive Nearest Neighbor classification using
Trevor Hastie's original Fortran implementation. DANN adapts the
distance metric based on local discriminant analysis to improve nearest
neighbor classification, particularly when classes have different
covariance structures or when irrelevant features are present.
