# dann2

dann2

## Usage

``` r
dann2(
  x,
  testx = NULL,
  y,
  k = 5,
  kmetric = NULL,
  epsilon = 1,
  rate = 0.5,
  fullw = FALSE,
  scalar = FALSE,
  iter = 1,
  covmin = 1e-04,
  cv = FALSE
)
```

## Arguments

- x:

  xmat

- testx:

  test mat

- y:

  labels

- k:

  nearest neighbors

- kmetric:

  kmet

- epsilon:

  epsilon

- rate:

  rate

- fullw:

  fullw

- scalar:

  scalar

- iter:

  iter

- covmin:

  covmin

- cv:

  cv

## Value

A list with items including Name of the Application, No. of pages
remaining (given the money), No. of fields remaining (given the money),
and when the application credits expire.
