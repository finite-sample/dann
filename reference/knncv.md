# knncv

Cross-validated k-means

## Usage

``` r
knncv(x, y, k = 5)
```

## Arguments

- x:

  matrix

- y:

  labels

- k:

  number of clusters

## Value

list of standardized matrices along with scaling information (centers)

## Examples

``` r
if (FALSE) { # \dontrun{
knncv(x, y, k=5)
} # }
```
